

\# 🛡️ life-immunity - OpenClaw 免疫系统插件



> 给AI穿上钢铁盔甲——安全审计、权限控制、沙箱隔离



\[!\[npm version](https://img.shields.io/npm/v/openclaw-plugin-life-immunity.svg)](https://www.npmjs.com/package/openclaw-plugin-life-immunity)

\[!\[npm downloads](https://img.shields.io/npm/dm/openclaw-plugin-life-immunity.svg)](https://www.npmjs.com/package/openclaw-plugin-life-immunity)



\## ✨ 特性



\- ✅ \*\*启动自动审计\*\*：每次启动自动检查权限和安全配置

\- ✅ \*\*高危操作拦截\*\*：实时检测并拦截 `rm -rf`、`format` 等危险命令

\- ✅ \*\*一键加固\*\*：`/harden` 命令自动修复常见安全问题

\- ✅ \*\*安全审计\*\*：`/audit` 命令运行官方安全检查

\- ✅ \*\*沙箱配置\*\*：生成安全的 Docker 沙箱配置建议

\- ✅ \*\*状态监控\*\*：`/immunity-status` 查看免疫系统状态

npm install -g openclaw-plugin-life-immunity
